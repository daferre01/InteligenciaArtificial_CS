BUCLE WHILE 🔁

El bucle while en Python es una estructura de control que permite repetir un bloque de código mientras una condición sea verdadera. La estructura básica del bucle while es la siguiente:

while condición:
    # código a ejecutar mientras la condición sea verdadera

La condición se evalúa al comienzo de cada iteración. Si la condición es verdadera, el código dentro del bucle se ejecuta. Después de cada iteración, la condición se vuelve a evaluar. Si la condición sigue siendo verdadera, el ciclo continúa. Si la condición se vuelve falsa, el bucle se detiene y la ejecución continúa con el código después del bucle.

n = 1
while n < 5:
    print(n)
    n += 1

En este ejemplo, el bucle while se ejecutará mientras el valor de n sea menor que 5. En cada iteración, se imprimirá el valor de n y luego se incrementará en 1. El bucle se detendrá cuando n sea igual o mayor que 5

BUCLE FOR 🔁

El bucle for en Python se utiliza para iterar sobre una secuencia (como una lista, una cadena de texto, un rango numérico, etc.) y ejecutar un bloque de código para cada elemento de la secuencia.

La estructura básica del bucle for es la siguiente:

for elemento in secuencia:
    # código a ejecutar para cada elemento

En cada iteración del bucle, el elemento actual de la secuencia se asigna a la variable elemento, y luego se ejecuta el código dentro del bucle.

Por ejemplo, si queremos imprimir cada elemento de una lista, podemos usar un bucle for de la siguiente manera:

lista = [1, 2, 3, 4, 5]

for elemento in lista:
    print(elemento)

En este caso, el bucle for se ejecutará cinco veces, una vez para cada elemento de la lista. En cada iteración, se imprimirá el valor del elemento correspondiente.

El bucle for también puede usar la función range() para generar una secuencia numérica. Por ejemplo:

for i in range(5):
    print(i)

En este caso, el bucle for se ejecutará cinco veces, una vez para cada número del 0 al 4. El valor de i se actualiza en cada iteración, comenzando desde 0 y aumentando en 1 en cada iteración.

El bucle for es una herramienta poderosa para recorrer y procesar elementos de una secuencia de manera eficiente en Python.

LISTAS🧾

Definición de listas en Python

Una lista en Python es una colección ordenada y mutable de elementos. Se pueden almacenar diferentes tipos de datos en una lista, como números, cadenas de texto, booleanos e incluso otras listas. Las listas se definen utilizando corchetes [ ] y separando los elementos con comas.

lista = [1, "dos", True, 3.14, [4, 5, 6]]

Principales métodos de listas en Python

  1. append(): Agrega un elemento al final de la lista.
    lista.append("nuevo elemento")
  1. insert(): Inserta un elemento en una posición específica de la lista.
    lista.insert(2, "elemento insertado")
  1. remove(): Elimina el primer elemento de la lista que coincida con el valor dado.
    lista.remove("elemento a eliminar")
  1. pop(): Elimina y devuelve el elemento en la posición especificada de la lista. Si no se especifica una posición, se elimina y devuelve el último elemento.
    elemento = lista.pop(1)
  1. clear(): Elimina todos los elementos de la lista.
    indice = lista.index("elemento a eliminar")

  1. index(): Devuelve el índice del primer elemento que coincida con el valor dado.
    indice = lista.index("elemento a buscar")
  1. count(): Devuelve la cantidad de veces que aparece un elemento en la lista.
    cantidad = lista.count("elemento a contar")
  1. sort(): Ordena los elementos de la lista de forma ascendente.
    lista.sort()
  1. reverse(): Invierte el orden de los elementos en la lista.
    lista.reverse()
  1. copy(): Crea una copia superficial de la lista.
    nueva_lista = lista.copy()

LISTAS POR COMPRENSIÓN 🧾

Las listas por comprensión, también conocidas como listas comprehensivas o listas generadas, son una forma concisa y poderosa de crear listas en Python. Permiten combinar un bucle for con una expresión dentro de corchetes [ ] para generar una lista de forma rápida y sencilla.

La sintaxis básica de una lista por comprensión es la siguiente:

[nombre_variable for elemento in secuencia]

Donde nombre_variable es el nombre de la variable que se utilizará para cada elemento de la secuencia, elemento es el elemento actual de la secuencia y secuencia es la secuencia de la cual se generarán los elementos de la lista.

Por ejemplo, si queremos generar una lista con los cuadrados de los números del 1 al 5, podemos usar una lista por comprensión de la siguiente manera:

cuadrados = [x**2 for x in range(1, 6)]

En este caso, la variable x toma los valores del 1 al 5 en cada iteración del bucle for, y se genera un elemento de la lista por cada valor de x elevado al cuadrado.

Las listas por comprensión también pueden incluir condiciones para filtrar los elementos de la secuencia. Por ejemplo, si queremos generar una lista con los números pares del 1 al 10, podemos usar una lista por comprensión con una condición if de la siguiente manera:

pares = [x for x in range(1, 11) if x % 2 == 0]

En este caso, solo se generan elementos de la lista para aquellos números x que cumplan la condición x % 2 == 0, es decir, que sean divisibles por 2.
