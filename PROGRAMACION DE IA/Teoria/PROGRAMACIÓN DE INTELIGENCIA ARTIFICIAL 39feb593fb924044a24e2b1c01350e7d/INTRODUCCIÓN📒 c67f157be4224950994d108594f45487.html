BUCLE WHILE üîÅ

El bucle while en Python es una estructura de control que permite repetir un bloque de c√≥digo mientras una condici√≥n sea verdadera. La estructura b√°sica del bucle while es la siguiente:

while condici√≥n:
    # c√≥digo a ejecutar mientras la condici√≥n sea verdadera

La condici√≥n se eval√∫a al comienzo de cada iteraci√≥n. Si la condici√≥n es verdadera, el c√≥digo dentro del bucle se ejecuta. Despu√©s de cada iteraci√≥n, la condici√≥n se vuelve a evaluar. Si la condici√≥n sigue siendo verdadera, el ciclo contin√∫a. Si la condici√≥n se vuelve falsa, el bucle se detiene y la ejecuci√≥n contin√∫a con el c√≥digo despu√©s del bucle.

n = 1
while n < 5:
    print(n)
    n += 1

En este ejemplo, el bucle while se ejecutar√° mientras el valor de n sea menor que 5. En cada iteraci√≥n, se imprimir√° el valor de n y luego se incrementar√° en 1. El bucle se detendr√° cuando n sea igual o mayor que 5

BUCLE FOR üîÅ

El bucle for en Python se utiliza para iterar sobre una secuencia (como una lista, una cadena de texto, un rango num√©rico, etc.) y ejecutar un bloque de c√≥digo para cada elemento de la secuencia.

La estructura b√°sica del bucle for es la siguiente:

for elemento in secuencia:
    # c√≥digo a ejecutar para cada elemento

En cada iteraci√≥n del bucle, el elemento actual de la secuencia se asigna a la variable elemento, y luego se ejecuta el c√≥digo dentro del bucle.

Por ejemplo, si queremos imprimir cada elemento de una lista, podemos usar un bucle for de la siguiente manera:

lista = [1, 2, 3, 4, 5]

for elemento in lista:
    print(elemento)

En este caso, el bucle for se ejecutar√° cinco veces, una vez para cada elemento de la lista. En cada iteraci√≥n, se imprimir√° el valor del elemento correspondiente.

El bucle for tambi√©n puede usar la funci√≥n range() para generar una secuencia num√©rica. Por ejemplo:

for i in range(5):
    print(i)

En este caso, el bucle for se ejecutar√° cinco veces, una vez para cada n√∫mero del 0 al 4. El valor de i se actualiza en cada iteraci√≥n, comenzando desde 0 y aumentando en 1 en cada iteraci√≥n.

El bucle for es una herramienta poderosa para recorrer y procesar elementos de una secuencia de manera eficiente en Python.

LISTASüßæ

Definici√≥n de listas en Python

Una lista en Python es una colecci√≥n ordenada y mutable de elementos. Se pueden almacenar diferentes tipos de datos en una lista, como n√∫meros, cadenas de texto, booleanos e incluso otras listas. Las listas se definen utilizando corchetes [ ] y separando los elementos con comas.

lista = [1, "dos", True, 3.14, [4, 5, 6]]

Principales m√©todos de listas en Python

  1. append(): Agrega un elemento al final de la lista.
    lista.append("nuevo elemento")
  1. insert(): Inserta un elemento en una posici√≥n espec√≠fica de la lista.
    lista.insert(2, "elemento insertado")
  1. remove(): Elimina el primer elemento de la lista que coincida con el valor dado.
    lista.remove("elemento a eliminar")
  1. pop(): Elimina y devuelve el elemento en la posici√≥n especificada de la lista. Si no se especifica una posici√≥n, se elimina y devuelve el √∫ltimo elemento.
    elemento = lista.pop(1)
  1. clear(): Elimina todos los elementos de la lista.
    indice = lista.index("elemento a eliminar")

  1. index(): Devuelve el √≠ndice del primer elemento que coincida con el valor dado.
    indice = lista.index("elemento a buscar")
  1. count(): Devuelve la cantidad de veces que aparece un elemento en la lista.
    cantidad = lista.count("elemento a contar")
  1. sort(): Ordena los elementos de la lista de forma ascendente.
    lista.sort()
  1. reverse(): Invierte el orden de los elementos en la lista.
    lista.reverse()
  1. copy(): Crea una copia superficial de la lista.
    nueva_lista = lista.copy()

LISTAS POR COMPRENSI√ìN üßæ

Las listas por comprensi√≥n, tambi√©n conocidas como listas comprehensivas o listas generadas, son una forma concisa y poderosa de crear listas en Python. Permiten combinar un bucle for con una expresi√≥n dentro de corchetes [ ] para generar una lista de forma r√°pida y sencilla.

La sintaxis b√°sica de una lista por comprensi√≥n es la siguiente:

[nombre_variable for elemento in secuencia]

Donde nombre_variable es el nombre de la variable que se utilizar√° para cada elemento de la secuencia, elemento es el elemento actual de la secuencia y secuencia es la secuencia de la cual se generar√°n los elementos de la lista.

Por ejemplo, si queremos generar una lista con los cuadrados de los n√∫meros del 1 al 5, podemos usar una lista por comprensi√≥n de la siguiente manera:

cuadrados = [x**2 for x in range(1, 6)]

En este caso, la variable x toma los valores del 1 al 5 en cada iteraci√≥n del bucle for, y se genera un elemento de la lista por cada valor de x elevado al cuadrado.

Las listas por comprensi√≥n tambi√©n pueden incluir condiciones para filtrar los elementos de la secuencia. Por ejemplo, si queremos generar una lista con los n√∫meros pares del 1 al 10, podemos usar una lista por comprensi√≥n con una condici√≥n if de la siguiente manera:

pares = [x for x in range(1, 11) if x % 2 == 0]

En este caso, solo se generan elementos de la lista para aquellos n√∫meros x que cumplan la condici√≥n x % 2 == 0, es decir, que sean divisibles por 2.
