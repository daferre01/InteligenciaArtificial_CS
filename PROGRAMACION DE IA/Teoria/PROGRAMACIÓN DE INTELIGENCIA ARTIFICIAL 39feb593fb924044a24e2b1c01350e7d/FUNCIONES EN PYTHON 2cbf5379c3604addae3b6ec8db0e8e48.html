FUNCIONES EN PYTHON

Definición de funciones en Python:

Una función en Python es un bloque de código reutilizable que realiza una tarea específica. Se define usando la palabra clave def, seguida del nombre de la función y paréntesis que pueden contener argumentos. A continuación, se muestra la sintaxis básica:

def nombre_de_funcion(argumento1, argumento2):
    # Código de la función
    return resultado

Ejemplo de definición de función:

def suma(a, b):
    resultado = a + b
    return resultado

En este ejemplo, hemos definido una función llamada suma que toma dos argumentos, a y b, y devuelve la suma de estos dos números.

Aconsejado al hacer bucles en funciones

def cuadrado(lista):
  lista2 = []
  for i in lista:
    lista2.append(i**2)
  return lista2

l = [1,2,3]
l = cuadrado(l)
print(l)

Parámetro *args:

Permite pasar un numero de argumentos variable de 1 a infinito

def sumar(*args):
    s=0 
    for i in args:
        s+=i
    return s

a= sumar(2,3,4,5)
print(a)

Parámetro **kargs

Se le pasan un numero variable de diccionarios de 1 a infinito

def sumar(**kargs):
    s=0
    for k,v in kargs.items():
        s+= v
    return s

s = sumar(naranjas=3, manzanas=4, uvas=3)
print(s)

EJEMPLO MEZCLADO DE ARG Y KARGS

En el siguiente ejemplo sumamos diccionarios y argumentos por igual

def una_funcion(*args, **kargs):
    s=0 
    for i in args:
        s+=i
    for k,v in kargs.items():
        s+= v
    return s

s= una_funcion(2,3,4,5,6,pepe=1,manuel=2,luis=7)
print(s)

FUNCIONES LAMDA

Son útiles cuando necesitas una función simple que se utiliza solo una vez y no requiere una definición formal con def

saludo = lambda: "Hola, mundo"
print(saludo())  # Imprime "Hola, mundo"

LAMBDA CON PARAMETROS

sumar=lambda a,b: a+b
print(sumar(1,3))

LAMBDA UTILIDAD EN UN SORTED PARA ESTABLECER ORDENACION

l = [(1,2), (3,2), (5,1)]
sorted(l, key=lambda k: k[1])
