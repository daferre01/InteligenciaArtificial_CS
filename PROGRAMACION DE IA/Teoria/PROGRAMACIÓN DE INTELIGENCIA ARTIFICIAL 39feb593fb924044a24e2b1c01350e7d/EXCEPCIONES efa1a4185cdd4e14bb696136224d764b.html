EXCEPCIONES

Una excepción es un evento que ocurre durante la ejecución de un programa y que interrumpe el flujo normal de instrucciones. Cuando una situación excepcional ocurre, se genera una excepción, y el control del programa se transfiere a un bloque de código especial llamado "manejador de excepciones". Este bloque maneja la excepción y puede tomar medidas adecuadas, como imprimir un mensaje de error, registrar la excepción, o realizar alguna otra acción específica.

def inversa(x):
  inv = 1/x
  print(f"la inversa de {x} es {inv}")

def f1(x):
  inversa(x)

def f2(x):
  f1(x)

def f3(x):
  try:
    f2(x)
  except Exception:
    print("Algo fue mal....")

Value Error

Error de un valor no valido

def inversa(x):
  inv = 1/x
  print(f"la inversa de {x} es {inv}")

def f1(x):
  inversa(x)

def f2(x):
  f1(x)

def f3(x):
  try:
    f2(x)
  except ValueError:
    print(("Ese valor es inválido!!!"))
  except Exception:
    print("Algo fue mal....")

Zero Division Error

No se puede dividir entre cero 

def inversa(x):
  inv = 1/x
  print(f"la inversa de {x} es {inv}")

def f1(x):
  inversa(x)

def f2(x):
  f1(x)

def f3(x):
  try:
    f2(x)
  except ZeroDivisionError:
    print(("No puedo dividir por 0!!!"))
  except ValueError:
    print(("Ese valor es inválido!!!"))
  except Exception:
    print("Algo fue mal....")

Raise

El metodo raise sirve para lanzar una excepcion directamente al ocurrir una accion determinada.
